# Strategic Communications Portfolio

This is a responsive, one-page portfolio starter that can be published for free
with GitHub Pages, Netlify, or another static website host.

## Start Here

1. Open `index.html` in a browser.
2. Search `index.html` for placeholder language such as `Your Name`, `Replace
   this paragraph`, and `Add`.
3. Replace the sample project text with your own work.
4. Put your resume and project files inside the `assets` folder.
5. Change each `href="#"` link so it points to the correct file or page.
6. Proofread every page and test every link before submitting.

## Required Content Checklist

- [ ] Name, professional headline, bio, skills, and contact details
- [ ] Downloadable resume
- [ ] At least one testimonial
- [ ] At least two individual assignments
- [ ] At least two group campaign projects
- [ ] At least one media item such as a video, graphic, infographic, or logo
- [ ] Campaign Sprint 4 crisis communication content
- [ ] Five short thought-leadership or reflection articles
- [ ] Challenge, strategy, execution, personal role, and outcomes for case studies
- [ ] Publicly accessible link tested in a private/incognito browser window

## Case Study Writing Formula

For each campaign, answer:

- **Context:** What organization, situation, or audience was involved?
- **Challenge:** What communication problem needed to be solved?
- **Insight:** What research or observation shaped the approach?
- **Strategy:** What audience, message, and channels did you prioritize?
- **Execution:** What did the team make or recommend?
- **Contribution:** What did you personally do?
- **Outcome:** What happened, what feedback did you receive, or how would success
  be measured?
- **Reflection:** What would you improve next time?

## Suggested Asset Names

Use clear file names without spaces:

```text
assets/
  your-resume.pdf
  campaign-one-plan.pdf
  campaign-one-social-post.png
  campaign-two-presentation.pdf
  crisis-communication-plan.pdf
  press-release.pdf
  stakeholder-memo.pdf
```

Do not publish private client information, private contact details, or group work
without identifying your own role and crediting teammates.
